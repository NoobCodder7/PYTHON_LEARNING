## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/python-learning.git
```

2. Navigate to the project folder:

```bash
cd python-learning
```

3. Run any Python file:

```bash
python filename.py
```

> Make sure Python 3.x is installed on your system.
